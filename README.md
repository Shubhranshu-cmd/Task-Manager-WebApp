# Task Manager Web App

A simple and responsive **Task Manager web application** built using **HTML, CSS, and JavaScript**.

This project allows users to manage their daily tasks directly in the browser with automatic saving using localStorage.

## Features

- Add new tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All / Active / Completed)
- Clear completed tasks
- Auto-save using browser localStorage
- Export tasks to JSON
- Import tasks from JSON
- Responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Browser LocalStorage API

## How It Works

The application stores all tasks locally in the browser using **localStorage**, allowing users to refresh the page without losing their tasks.

## How to Run

1. Download the repository
2. Open `index.html` in your browser

or run directly via **GitHub Pages**.

## Project Purpose

This project was created as a **learning project to demonstrate basic front-end development skills**, including DOM manipulation, UI design, and client-side storage.
